## Cat Quest
In the game, you need to find a way out of the cave by passing through all areas of the map.

Technologies:
- Tomcat (embedded)
- Servlet, JSP, JSTL, JDBC
- H2 Database (embedded)

Features:
- Map generation
- Moving around the map
- Loading/Saving the game state
